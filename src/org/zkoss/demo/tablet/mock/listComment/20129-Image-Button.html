Hi I have an issue when creating a button with a image as the background, the size of the property on the button has no effect when placed inside a <hlayout> / <hbox>

Here is my code below:

<zk xmlns:w="client">
	<style>
		.mybtn{background-image: url('./layout/3.png');}
		.myborder{background-image: url('./layout/bck.png');}
	</style>
	<hlayout height="100%" width="100%">
		<vlayout id="mainvlayout" spacing="1px">
			This is the first block of hlayout
			<button label="Button 1-0"></button>
			<hlayout spacing="1px">
				<button id="btn1" label="Button 2-0"></button>
				<button id="btn2" label="Button 2-1"></button>
				<button id="buttonMenu1"
					w:onClick="jq(this.$f('buttonMenu1')).fadeTo('slow',0.6); jq(this.$f('buttonMenu2')).fadeTo('slow',1.0);"
					width="50%" height="50%" sclass="mybtn" label=" ">
				</button>
				<button id="buttonMenu2"
					w:onClick="jq(this.$f('buttonMenu2')).fadeTo('slow',0.6); jq(this.$f('buttonMenu1')).fadeTo('slow',1.0);"
					width="50%" height="50%" sclass="mybtn" />
			</hlayout>
		</vlayout>
	</hlayout>
</zk>

The problem is that id=btn1 and id= btn2 with text works perfect, but id=buttonMenu1 and id=buttonMenu2 without text gives the result of a button the size of 1px by 1px

Any help would be appreciated.
