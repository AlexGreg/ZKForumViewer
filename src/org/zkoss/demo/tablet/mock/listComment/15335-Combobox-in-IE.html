Using ZUL in ZK 6.0.1

No combobox attributes used in controller only onSelect event to retrieve values.

<combobox id="ModelCombo" model="@load(vmCreationOuvrage.labelListModel)" selectedItem="@bind(vmCreationOuvrage.currentModelOuvrage)">
     <template name="model">
          <comboitem label="@load(each.description)" />
     </template>
</combobox>
