Hi,

A workaround is call javascript function to set it, please refer to the sample:

<zk xmlns:w="client">
  <button label="add marker and set max zoom">
    <attribute name="onClick">
      Gmarker m = new Gmarker();
      m.setLat(23);
      m.setLng(121);
      m.setParent(gmaps);
      String cmd = "setTimeout(function () { var m = zk.Widget.$('#"+m.getUuid()+"');"
                                          + "m.setMaxzoom(21);}, 2000)";
      Clients.evalJavaScript(cmd);
    </attribute>
  </button>

  <gmaps id="gmaps" width="800px" height="800px" lat="23" lng="121" showLargeCtrl="true">

  </gmaps>
</zk>

You can also try the gmapsz 3.0, it is available at google code and no such issue:
http://code.google.com/p/zkgmapsz/downloads/list

Regards,
Ben
