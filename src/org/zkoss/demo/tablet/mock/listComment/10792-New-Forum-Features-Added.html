New Forum Features Added!

wingchan
29 Dec 2009 21:12:18 GMT
29 Dec 2009 21:12:18 GMT

Hi everyone,

In a continuing effort to make ZK forum even better, we have released another forum upgrade:

1) NEW Feature: Advanced search

To use the advanced search options, please press the Advanced Search button on the top-right corner of the forum. You will be presented with a form making advanced searching easier! This form enables you to find forum content by specifying one or more search terms. You can search by author and date, and then order the results by date, number of posts and number of thread views.

2) NEW Feature: Email Notification

To ensure that you receive the latest replies to your original thread, we have added a new feature which will notify you by email when someone replies.

If you'd like to be notified by email when someone replies, please use the following instructions:

* Go to you User Profile by clicking your user name in the top right corner or click this link.
* Click the “Preferences” tab
* Select “Yes” to the question “Would you like to be notified of thread replies by email?”
* Save your changes.

We have only highlighted two major feature improvements. However, we have also implemented many other “under the hood” improvements to make your ZK forum experience better than ever.

Please try the new features and let us know if you observe any strange behavior!


Regards,
Wing
The ZK Team
